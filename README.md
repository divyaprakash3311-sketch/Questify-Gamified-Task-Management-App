# Questify — Gamified Task Manager

Questify is a full-stack task management app that turns your to-dos into quests. Complete tasks, earn XP, build streaks, unlock badges, and level up.

## Features

- Login and sign up
- Create, edit, delete tasks
- Real-time updates
- Responsive design
- XP, streaks, badges, and boss tasks

## Tech Stack

- React / Next.js
- Node.js / Express
- MongoDB / PostgreSQL
- JWT
- Socket.IO
- Tailwind CSS

## Setup

```bash
npm install
npm run dev
```

## Idea

Turn everyday tasks into a game and stay productive while leveling up.
